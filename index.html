O que é semântica no HTML?

Semântica no HTML refere-se ao uso de elementos HTML que transmitem o significado ou a estrutura do conteúdo, em vez de simplesmente fornecer uma formatação visual. A ideia é usar tags que representem a intenção e o papel do conteúdo na página, tornando o código mais compreensível tanto para desenvolvedores quanto para máquinas (como motores de busca e leitores de tela).

Estrutura web

A estrutura HTML fornece a base para criar e organizar páginas na web, estabelecendo a forma como o conteúdo é exibido e interpretado pelos navegadores. HTML, que significa HyperText Markup Language, usa uma série de tags e elementos para definir a estrutura e o layout de um documento.
A principal organização que foca na padronização para a criação e a interpretação de conteúdos para a Web é a World Wide Web Consortium, mais conhecida como W3C


Estrutura Semântica em HTML:

Tags Semânticas em HTML

As tags semânticas no HTML são elementos que ajudam a definir a estrutura e o significado do conteúdo de uma página web. Elas não são apenas para aparência, mas servem para descrever a função do conteúdo, o que torna a página mais acessível e facilita a compreensão tanto para os navegadores quanto para os motores de busca.

Exemplos de Tags no HTML:

Tag < header >


O elemento < header > serve para definir o cabeçalho de um documento ou seção em HTML. Dentro deste elemento, é possível inserir desde cabeçalhos (< h1 > a < h6 >) até imagens, parágrafos e listas de navegação. Veja a seguir um exemplo de como utilizar o < header >: 

#Tags header <Exemplo>
    < header >  
        < h1 > Title Pagina < h1 >
        < h2 > Conteúdo da pagina < h2 >
    < /header >

Tag < section >


Define uma seção genérica dentro de um documento, geralmente utilizada para agrupar conteúdos que estão tematicamente relacionados. Essa organização pode ser indicada através de elementos de cabeçalho, como < h1 > a < h6 >. O elemento < section >, por exemplo, pode ser empregado para descrever diferentes seções ou tópicos de um documento. Aqui está um exemplo de como usar o < section >: 

#Tags < section >  <Exemplo>
    < section >  
        
        < h3 > Sesão 1 < h1 >


        < p > Um texto aleatorio a ser adicionado aqui ... < /p  >
    < /section >

Tag < article >


Utilizamos o elemento < article > para declarar um conteúdo que pode se entender de forma independente dentro de um documento HTML, como um artigo de blog. É recomendável que cada < article > seja identificado por um título. Veja a seguir um exemplo de como usar o < article >: 

#Tags < article >  <Exemplo>
    < article >  
        
        < h3 > Titulo Artigo 1 < h3 >

        < p > Um texto aleatorio a ser adicionado aqui ... < /p  >
    < /article > 
        
            < article >  
                
                < h3 > Titulo Artigo 2 < h3 >
        
                < p > Um texto aleatorio a ser adicionado aqui ... < /p  >
            < /article >

Tag < nav >


O elemento < nav > é utilizado para agrupar links de navegação dentro de um documento. Esses links são geralmente criados com os elementos < ul >, < li >, e < a >. Veja a seguir um exemplo de como usar o < nav >: 

#Tags < nav >  <Exemplo>
     < nav >  
         
         < li > < a href=""> Pagina 1  < /li > < /a >
         < li > < a href=""> Pagina 2  < /li > < /a >
         < li > < a href=""> Pagina 3  < /li > < /a >
     < nav >

Tag < main >


O elemento < main > define o conteúdo principal e mais relevante de uma página. Para garantir uma estrutura adequada, uma página deve ter apenas um < main >. Veja a seguir um exemplo de como utilizar o < main >: 

#Tags < main >  <Exemplo>
     < main >  
         
         < h2 > Titulo Artigo 1 < h2 >
 
         < p > Um texto aleatorio a ser adicionado aqui ... < /p  >

         < article >  
                 
            < h3 > Titulo Artigo 2 < h3 >
    
            < p > Um texto aleatorio a ser adicionado aqui ... < /p  >
        < /article > 
     < /main >

Tag < figure >


O elemento < figure > é usado para inserir uma figura e pode incluir uma descrição com o elemento < figcaption >. Aqui está um exemplo de uso do < figure >: 

#Tags < figure >  <Exemplo>
             < figure >  
                 
                 < img src="" alt="" >
                 < figcaption > descrição da Imagem< /figcaption> >
         
             < /figure >

Tag < footer >


O elemento < footer > define o rodapé de um documento, geralmente localizado no final de uma página web. Ele é utilizado para fornecer informações como autoria, contato do autor e data de criação do conteúdo. Veja um exemplo de como usar o < footer >:
