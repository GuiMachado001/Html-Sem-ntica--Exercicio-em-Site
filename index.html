O que é semântica no HTML?

Semântica no HTML refere-se ao uso de elementos HTML que transmitem o ​ significado ou a estrutura do conteúdo, em vez de simplesmente fornecer uma ​ formatação visual. A ideia é usar tags que representem a intenção e o papel do​ conteúdo na página, tornando o código mais compreensível tanto para ​ desenvolvedores quanto para máquinas (como motores de busca e leitores de​ tela).

Tags Semânticas em HTML

As tags semânticas no HTML são elementos que ajudam a definir a estrutura e o significado do conteúdo de uma página web. Elas não são apenas para aparência, mas servem para descrever a função do conteúdo, o que torna a página mais acessível e facilita a compreensão tanto para os navegadores quanto para os motores de busca.

Exemplos de Tags no HTML:

Tag < header >


O < header > é utilizado para representar o cabeçalho de um documento ou seção declarado no HTML. Nele podemos inserir elementos de < h1 > a < h6 >, até elementos para representar imagens, parágrafos ou mesmo listas de navegação. Exemplo de uso de < header >: 

#Tags header <Exemplo>
    < header >  
        < h1 > Title Pagina < h1 >
        < h2 > Conteúdo da pagina < h2 >
    < /header >

Tag < section >


Define uma seção genérica de um documento. É geralmente usada para agrupar conteúdos tematicamente relacionados., o qual é definido por meio de um dos elementos entre < h1 > e < h6 >. Podemos utilizar o < section >, por exemplo, para descrever as seções/tópicos de um documento. Exemplo de uso de < section >: 

#Tags < section >  <Exemplo>
    < section >  
        
        < h3 > Sesão 1 < h1 >


        < p > Um texto aleatorio a ser adicionado aqui ... < /p  >
    < /header >
